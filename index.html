PWM Control

Control brightness and speed with PWM signals.

PWM LED Brightness Control:

int ledPin = 9; // PWM pin

void setup() {
  pinMode(ledPin, OUTPUT);
}

void loop() {
  for (int i = 0; i <= 255; i++) {
    analogWrite(ledPin, i);
    delay(10);
  }
  for (int i = 255; i >= 0; i--) {
    analogWrite(ledPin, i);
    delay(10);
  }
}

PWM Pins: 3, 5, 6, 9, 10, 11 (Arduino Uno)

Button Input & Debouncing

Learn to read button inputs and handle debouncing.

Button Input with Debouncing:

const int buttonPin = 2;
const int ledPin = 13;
int buttonState = 0;
int lastButtonState = 0;
unsigned long lastDebounceTime = 0;
const unsigned long debounceDelay = 50;

void setup() {
  pinMode(buttonPin, INPUT);
  pinMode(ledPin, OUTPUT);
}

void loop() {
  int reading = digitalRead(buttonPin);
  
  if (reading != lastButtonState) {
    lastDebounceTime = millis();
  }
  
  if ((millis() - lastDebounceTime) > debounceDelay) {
    if (reading != buttonState) {
      buttonState = reading;
      if (buttonState == HIGH) {
        digitalWrite(ledPin, HIGH);
      } else {
        digitalWrite(ledPin, LOW);
      }
    }
  }
  
  lastButtonState = reading;
}

Key Concept: Debouncing prevents false readings from button bouncing

Time & Millis() Function

Master timing in Arduino without using delay().

Non-Blocking Timing Code:

unsigned long previousMillis = 0;
const unsigned long interval = 1000; // 1 second

void setup() {
  Serial.begin(9600);
}

void loop() {
  unsigned long currentMillis = millis();
  
  if (currentMillis - previousMillis >= interval) {
    previousMillis = currentMillis;
    Serial.println("1 second passed!");
  }
}

Advantage: Code runs other functions while timing

Sound & Buzzer Control

Generate tones and control buzzers with tone() function.

Playing Musical Notes:

const int buzzerPin = 9;

// Frequency of notes
#define NOTE_C4 262
#define NOTE_D4 294
#define NOTE_E4 330
#define NOTE_F4 349

void setup() {
  pinMode(buzzerPin, OUTPUT);
}

void loop() {
  // Play a melody
  playNote(NOTE_C4, 250);
  playNote(NOTE_D4, 250);
  playNote(NOTE_E4, 250);
  delay(500);
}

void playNote(int frequency, int duration) {
  tone(buzzerPin, frequency, duration);
  delay(duration + 50);
}

Usage: tone(pin, frequency, duration)

Arduino Pinout Reference

Complete pinout diagrams and specifications for different Arduino boards

Arduino Uno Pin Configuration

Digital Pins (0-13)

Pin 0 (RX) - Serial Input
Pin 1 (TX) - Serial Output
Pins 3, 5, 6, 9, 10, 11 - PWM Output
Pins 2, 4, 7, 8, 12, 13 - Standard Digital

Analog Pins (A0-A5)

A0-A5 - Analog Input (10-bit, 0-1023)
Voltage Range: 0V to 5V

Power Pins

5V - +5V Power Supply
3.3V - +3.3V Power Supply
GND - Ground (2 pins)
Vin - Power Input (7-12V)

Quick Reference Code:

// Digital Pin Setup
pinMode(13, OUTPUT);     // Set pin 13 as output
digitalWrite(13, HIGH);  // Set pin 13 HIGH

// Analog Pin Setup
int value = analogRead(A0); // Read from A0 (0-1023)

// PWM Output (pins 3,5,6,9,10,11)
analogWrite(9, 255);     // Full brightness on pin 9

// Serial Communication (pins 0,1)
Serial.begin(9600);
Serial.println("Hello");

Arduino Nano Pin Configuration

Digital Pins (0-13)

Same as Uno - 14 digital pins total
Pins 3, 5, 6, 9, 10, 11 - PWM capable
Pin 0 (RX) & Pin 1 (TX) - Serial

Analog Pins (A0-A7)

A0-A7 - Analog Input (8 pins available)
More analog pins than Uno!

Special Features

Smaller form factor (18x45mm)
Can be powered via USB or 5V pin
Mini USB connector

Arduino Mega 2560 Pin Configuration

Digital Pins (0-53)

54 digital pins total!
Pins 2-13, 44-46 - PWM capable (15 PWM pins)
Pins 0,1,19,18 - Multiple serial ports

Analog Pins (A0-A15)

A0-A15 - Analog Input (16 pins)
Great for projects with many sensors

Advanced Features

4 Hardware UART ports
54KB SRAM (vs 2KB on Uno)
256KB Flash memory

Arduino Pro Mini Pin Configuration

Digital Pins (0-13)

14 digital pins (same as Uno)
Pins 3, 5, 6, 9, 10, 11 - PWM
No USB - requires FTDI programmer

Analog Pins (A0-A6)

A0-A6 - Analog Input (7 pins)
A4, A5 - I2C pins (SDA, SCL)

Compact Design

Smallest Arduino (33x18mm)
Perfect for wearables and embedded projects
Available in 3.3V and 5V versions

‚ö° Pinout Tips & Best Practices

Analog Pins as Digital

You can use analog pins as digital pins: pinMode(A0, OUTPUT) and digitalWrite(A0, HIGH)

PWM Pins

Use analogWrite() only on PWM pins for analog output (0-255 maps to 0-5V)

Serial Pins

Avoid using pins 0 & 1 for other purposes when using Serial communication

Current Limits

Each pin can source/sink ~40mA. Total board ~200mA. Use external power for motors

Voltage Levels

Digital pins: 5V (or 3.3V on some boards). Analog input: 0-5V for 0-1023

Protection

Always use current-limiting resistors with LEDs and external circuits
